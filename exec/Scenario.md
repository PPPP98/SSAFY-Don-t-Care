### 2. 랜딩 페이지 (온보딩)

![온보딩 페이지](./videos/onboarding.gif)

```
화면에서 보시는 것처럼 랜딩 페이지에서는 서비스 소개와 서비스 사용법이 안내됩니다.
사용자는 로그인 후 바로 투자 의사결정을 돕는 AI 에이전트 기반 서비스를 체험할 수 있습니다.
```

---

### 3. 메인 페이지 (채팅 인터페이스 & 에이전트 목록)

![메인 페이지](./videos/mainpage.gif)

```
익숙한 채팅 방식의 인터페이스가 제공됩니다.
실제 활용할 수 있는 에이전트 목록은 우측 탭에서 확인할 수 있고, 실시간 데이터를 기반으로 동작하는 것을 확인할 수 있습니다.

질문을 입력하면 비서 실장이 질문 의도를 파악하고, 필요한 전문 비서들에게 업무를 분배합니다.
각 전문 비서들은 도구와 외부 API들을 활용해 실제로 정보를 수집하고 분석합니다.
마지막으로 비서 실장이 결과를 취합해 하나의 완성된 답변으로 사용자에게 전달합니다.
```

---

### 4. 에이전트 호출 과정

![에이전트 호출](./videos/agentcall.gif)

```
(리포트 요청 시나리오)
투자 의견 보고서를 요청하면 각 에이전트들이 호출됩니다.

뉴스 분석 비서: Google News API를 통해 최근 기사를 수집하고 감정 분석을 수행합니다.

재무 분석 비서: DART API에서 기업 재무 데이터를 가져와 주요 지표를 계산합니다.

시장 분석 비서: Yahoo Finance 데이터를 기반으로 RSI, MACD 등 기술적 지표를 분석합니다.

리스크 분석 비서: 자체 백테스트 엔진을 실행해 전략의 과거 성과를 시뮬레이션합니다.

이 모든 과정은 병렬로 진행되기 때문에 속도도 빠릅니다.
```

### 5. 투자 리포트 결과 화면

![답변 결과](./videos/result.gif)

```
최종적으로 비서 실장이 모든 결과를 종합하여 하나의 보고서 형태로 사용자에게 제공합니다.
데이터와 지표들은 시각화되어 보여지고, 참고한 출처들도 함께 제공됩니다.

이렇게 생성된 투자 보고서를 기반으로 사용자는 더 합리적인 의사결정을 내릴 수 있습니다.
```
